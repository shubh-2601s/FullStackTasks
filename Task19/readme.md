# Sample Git Project – Version Control Demonstration

## 📌 Project Overview

This project demonstrates basic Git operations including repository initialization, staging, committing, maintaining version history, and pushing to GitHub.

---

## 🛠️ Technologies Used

* Git
* GitHub
* Java (Sample Project)

---

## 📁 Project Structure

```
sample-git-project/
│── src/
│   └── Main.java
│── .gitignore
│── README.md
```

---

## ▶️ How to Run the Project

```
javac src/Main.java
java -cp src Main
```

---

## 🔧 Git Commands Used

### Initialize Repository

```
git init
```

### Check Status

```
git status
```

### Stage Files

```
git add .
```

### First Commit

```
git commit -m "Initial commit: Added project structure"
```

### Second Commit

```
git commit -m "Added main application logic"
```

### Third Commit

```
git commit -m "Updated README with project details"
```

### View Commit History

```
git log
```

### Add Remote Repository

```
git remote add origin https://github.com/your-username/sample-git-project.git
```

### Push to GitHub

```
git push -u origin main
```

### Make Changes and Push Again

```
git add .
git commit -m "Updated project files"
git push
```

## 🔄 Version Control Workflow

```
Working Directory → Staging Area → Local Repository → Remote Repository
```
